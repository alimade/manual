## 文档目标

本指南汇总 Smart 人形怪（战士/法师/道士）在“移动速度、攻击速度、施法节奏、AI 行为节流”上的生效规则。

适用对象：

- 需要做平衡调整的运营/策划/服主
- 需要定位“为什么怪物出手快慢不一致”的维护人员

说明：

- 外部 KEY：可在配置文件中调整的键名（通常在 \[Setup\] 段）。
- DB 属性名：技能数据中的字段名（非 Setup 键）。

## 全职业通用（基础攻速/走速）

| 项目 | 外部 KEY | DB 属性名 | 默认值 | 是否可自定义 | 描述 |
| --- | --- | --- | --- | --- | --- |
| 战士基础攻击间隔 | Setup.WarrAttackTick | - | 750 | 是 | 每次成功出手后都会进入攻击间隔，间隔未到不会触发下一次有效攻击。 |
| 法师基础攻击间隔 | Setup.WizardAttackTick | - | 1330 | 是 | 法师所有主动施法也受基础攻击节拍限制，未到间隔时会继续等待。 |
| 道士基础攻击间隔 | Setup.TaosAttackTick | - | 1350 | 是 | 道士的治疗、召唤、输出动作都先受基础攻击节拍约束。 |
| 战士移动间隔 | Setup.nWarrWalkTime | - | 600 | 是 | 数值越小，追击与贴身动作越紧凑。 |
| 法师移动间隔 | Setup.nWizardWalkTime | - | 600 | 是 | 数值越小，拉扯与脱离近身位的速度越快。 |
| 道士移动间隔 | Setup.nTaosWalkTime | - | 600 | 是 | 数值越小，边退边打与补位动作越频繁。 |

## 战士（TSmartWarr）专项延时

### 通用项（不区分智能档）

| 项目 | 外部 KEY | DB 属性名 | 默认值 | 是否可自定义 | 描述 |
| --- | --- | --- | --- | --- | --- |
| 烈火剑法间隔 | Setup.nHeroMagicBlazeTick | - | 7000 | 是 | 烈火属于爆发技能，间隔没到时不会重复进入可释放状态。 |
| 开天斩间隔 | Setup.LongSwordTime | - | 20000 | 是 | 开天斩触发后会进入长冷却窗口，窗口内主要依赖其他输出技能。 |
| 逐日剑法间隔 | Setup.LongFireHitSkillTime | - | 10000 | 是 | 逐日按固定冷却节奏释放，冷却未到时会回退其他攻击行为。 |
| 野蛮起手节流 | - | - | 3000（固定） | 否 | 短时间内不会连续尝试同类起手，避免动作抖动与打断普攻链。 |
| 智能档入口 | `SmartMob 文本: BaseInfo.IntelligenceLevel` | - | 中（默认） | 是（按怪名配置） | 决定战士使用低/中/高三档行为节奏（配置值分别是 `Low/Medium/High`）。 |

| 项目 | 外部 KEY | 默认值 | 是否可自定义 | 描述 |
| --- | --- | --- | --- | --- |

| 修位节流间隔 | `BaseInfo.IntelligenceLevel = 低（配置值 Low）` | 500ms | 否（固定） | 调位频率较低，更少频繁变向。 |
| 野蛮起手积极性 | `BaseInfo.IntelligenceLevel = 低（配置值 Low）` | 低（约 1/6 概率） | 否（固定） | 更少主动用高节奏起手，动作更“稳”。 |
| 开天倾向修正 | `BaseInfo.IntelligenceLevel = 低（配置值 Low）` | 基础权重下调 | 否（固定） | 更少触发开天偏好，整体爆发感更弱。 |

| 项目 | 外部 KEY | 默认值 | 是否可自定义 | 描述 |

## 法师（TSmartWizard）专项延时

| 项目 | 外部 KEY | DB 属性名 | 默认值 | 是否可自定义 | 描述 |
| --- | --- | --- | --- | --- | --- |
| 法师技能冷却（逐技能） | - | Magic.dwDelayTime | 取决于技能数据 | 是（改技能数据） | 法师每个技能独立记冷却，优先级再高也必须先过该技能自己的冷却。 |
| 火墙尝试节流 | - | - | 固定隔次尝试 | 否 | 火墙不会每个攻击周期都抢出手，主要用于补充压制。 |
| 法师修位节流 | - | - | 45（固定） | 否 | 撤位行为有最小间隔，避免连续改路径导致抖动。 |

补充：

- 当前法师 Smart 固定高智能，不区分低/中/高。

## 施法延时的配置边界（重点）

如果你只改 \[Setup\] 下的攻速/走速键：

- 会明显改变“动作节奏”（追击、平A、技能尝试频率）。
- 但不会直接改“单个技能自身冷却”。

如果你要改“法师/道士某个技能本身的释放频率”：

- 需要改技能数据中的 Magic.dwDelayTime（DB 属性）。
- 这属于技能层，不在 Setup.\* 全局键里。

## 调参建议（用户视角）

1. 先调基础节奏：WarrAttackTick / WizardAttackTick / TaosAttackTick 与三职业 WalkTime。
2. 再调战士爆发窗： HeroMagicBlazeTick、LongSwordTime、LongFireHitSkillTime。
3. 最后再调技能库：仅在需要改某个技能独立频率时再改 Magic.dwDelayTime。
