# 通用控制

## 简单使用

<img
  src="/images/image-4.png"
  alt="Image"
  style={{ width:"62%" }}
/>

封速主要分为间隔控制和并发量控制（又称封多倍），控制模式根据两种方式来做：

- 停顿 - 当触发之后，玩家将会进入停顿 3 秒（时间内置固定）
- 反弹 - 当触发之后，玩家将会出现卡刀现象
- 忽略 - 当触发之后，玩家的攻击表象是成功但没有实际伤害
- 脚本 - 更加灵活的模式处理方式，支持少量的变量（搭配脚本编辑使用），具体查看下面的脚本编辑示例

<Note>
  一般使用 “基本” 封速就可以了，“外挂” 选项中一般不需要启用
</Note>

## 速度监控

<img
  src="/images/image-8.png"
  alt="Image"
  style={{ width:"73%" }}
/>

当 M2Server 的游戏速度保存之后会同步至游戏网关(RunGate)，如果有玩家在线就可以查看相关的速度信息，这里查看的玩家速度信息都是实时的数据，如果间隔到 0 了说明玩家每刀都有效果，开了多倍之后数据量会非常庞大可能会导致 M2Server 处理不过来产生的其他玩家原地 “幻刀” 问题，注意合理设置。

## 脚本设置

```txt
[@IllegalHandle]
#IF
$HUM.Level > 100
#ACT
SendMsg 请不要进行非法操作，注意和谐社会!!! 255 252
ReturnFail
#ELSEACT
ReturnSkip
```

脚本处理中目前暂时只支持一些简单的人物变量判断和返回，参考如下：\
\
变量

- \$HUM.Level - 人物等级
- \$Hum.LuckyPoint - 幸运
- \$Hum.SpeedPoint - 速度
- \$Hum.AC1 - 防御低值
- \$Hum.AC2 - 防御高值
- \$Hum.MAC1 - 魔御低值
- \$Hum.MAC2 - 魔御高值
- \$Hum.DC1 - 攻击低值
- \$Hum.DC2 - 攻击高值
- \$Hum.MC1 - 魔法低值
- \$Hum.MC2 - 魔法高值
- \$Hum.SC1 - 道术低值
- \$Hum.SC2 - 道术高值
- \$Hum.HP - 人物血量
- \$Hum.MP - 人物蓝量

处理

- ReturnStop - 停顿处理，如: ReturnStop 3 (支持自定义秒)
- ReturnFail - 反弹处理
- ReturnSkip - 忽略处理
- SendMsg - 发送消息，注意这里和 M2Server 版本中的使用有点不一样

<Note>如果支持的不够请在 KOOK 中提需求，合理的话我们会慎重考虑进行扩展.</Note>

# 地图控制

地图控制用于按地图覆盖全局速度，策略是“全局基线 + 地图 ini 按键覆盖”。

## 地图写法

在 `Envir/MapInfo.txt` 的目标地图行增加：

```txt
[D717 霸王教主之家] FIGHT NORECONNECT(MAPMOVE) NOPOSITIONMOVE SPEED(沙巴克皇宫.ini)
```

## 文件路径

`SPEED(沙巴克皇宫.ini)` 对应文件：

```txt
Envir/Speed_Def/沙巴克皇宫.ini
```

## 生效时机

1. 玩家登录后下发当前地图速度配置。
2. 同服切图成功后实时重发速度配置。
3. 地图未配置 `SPEED` 时，完全使用全局速度。
4. 速度文件缺失或解析失败时，告警并回退全局，不阻断进图。

## 当前不覆盖项

地图控制仅覆盖布尔/数值字段，不覆盖 `*ControlScript` 脚本字段。
