# **使用预览**

![Image](/images/image-1.png)

当使用 “2.5.0-rc.3” 版本及以上的万能登陆器之后既可以登陆游戏使用快捷键 “Shift + Ctrl + B” 呼出右边侧边栏的简易界面编辑工具，该工具的配置更改之后既可以在场景中看到效果（你可以点击 “刷新” 或者 “Ctrl + S" 快捷键进行保持）

**不同分辨率下的加载规则**

- 800x600 - 直接加载 Data\\uiConfig.json
- 1024x768 - 直接加载 Data\\uiConfig1024.json
- 1280x800 - 直接加载 Data\\uiConfig1280.json
- 更大分辨率暂不支持

> 注意：上述的配置刚开始不需要存在，可以在打开简易编辑器的时候再编辑保存会自动生成！！！

# **配置大纲**

## **场景类型**

- Lib - 图库
- Width - 宽度 - 场景的绝对大小，如果场景大小和窗口大小不一致则会产生偏移影响 doAboslute 的子元素
- Height - 高度
- Background - 场景背景索引
- LayoutStyle
  - doAutoCenter - 水平垂直居中
  - doAbsolute - 靠左上绝对坐标

## **窗口类型**

- Lib - 图库
- Background - 暂时未使用，使用 "-1" 占位
- AutoSize
  - True - 使用系统内置的大小，忽略上述的宽度和高度
  - False - 使用上述自定义的宽度和高度作为大小
- LayoutStyle
  -  doAbsolute - 绝对坐标，根据场景的大小和偏移来放置
  - doAutoCenter - 根据父节点的宽高来进行自动水平垂直居中

## **文本框类型**

- Width - 宽度
- Height - 高度
- Left - 坐标 X
- Top - 坐标 Y
- AutoSize
  - True - 使用系统内置的大小，忽略上述的宽度和高度
  - False - 使用上述自定义的宽度和高度作为大小
- LayoutStyle
  - doAbsolute - 绝对坐标，根据场景的大小和偏移来放置
  - doAutoCenter - 根据父节点的宽高来进行自动水平垂直居中

## **标签类型**

- Left - 坐标 X
- Top - 坐标 Y
- LayoutStyle
  - doAbsolute - 绝对坐标，根据场景的大小和偏移来放置
  - doAutoCenter - 根据父节点的宽高来进行自动水平居中 - **(暂不支持垂直居中)**

## **按钮类型**

- Lib - 图库
- Background - 目前暂未使用
- Pressed - 按钮按下的图库索引
- Width - 宽度
- Height - 高度
- Left - 坐标 X
- Top - 坐标 Y
- AutoSize - 同上
- LayoutStyle - 同上

# **图库范围**

- WUISet1 - Data\\uiSet1.wil
- WUISet2 - Data\\uiSet2.wil
- WUISet3 - Data\\uiSet3.wil

- WProgUseX - Data\\ui\\Prguse.wil （未来会被弃用，不再推荐使用）

# **界面配置**

主要分为四个场景，每个场景的弹出框需要单独分开设置确保灵活性，每个场景下的具体界面如下述所示：

## **登录场景**

```json expandable
{
  "LoginSceneUIConfig": { 场景类型
    "Lib": "WUISet1",
    "Background": 0,
    "Width": 800,
    "Height": 600,
    "Left": 0,
    "Top": 0,
    "LayoutStyle": "doAutoCenter",
    "LoginUIConfig": { 查看下述的登陆界面 }
    "RegisterUIConfig": { 查看下述的注册界面 }
    "PwdChangeUIConfig": { 查看下述的密码修改界面 }
    "MsgDialogBigHorizontalUIConfig": { 弹出框的面板设置 }
  }
}
```
